    TIPS:
    * Single clicks in the chart only.
    * Use the arrows and "enter" to move by pixel when calibrating the chart or settings points
    * Use +/- keys to change the zoom factor of the marking window

    How to use this tool:
    1. Load your graph
    2. Set the datum and mark the matching measured values
    3. Set a known calibration point with a differeny x,y and enter the measured values
    4. Select "Mark points" and calibrate the chart against some points from your curve.
    5. Adjust regression degree until it fits the curve
    6. Test out the prediction prior to using the regression formula elshhwere
